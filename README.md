# wttr.in-script
Script to check the weather via wttr.in

***Make sure the Script is executeable:***

```chmod +x wttr```

***Move to /bin/ (or somewhere in $PATH):***

```mv wttr /bin/.```

***Execute:***

Syntax wttr <location> <lang>

```wttr karlsruhe de```
